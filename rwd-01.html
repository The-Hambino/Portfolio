Responsive Web Design - Promotional Landing Page Template

Responsive Template

My Role: Create template from scratch using exisiting header and footer.

This responsive template was created for a team of 20+ designers to use in conjunction with 3rd party vendor assets. Vendor assets would come in many different forms, so this template needed to be flexible enough to accomodate many different scenarios while keeping production turnaround time quick. Template includes:

  • Preset layout options: 80/20, 66/33, and 50/50 splits.
  • Flexible images optimized for both desktop and mobile.
  • Responsive video containers.
  • Product cells with dynamic pricing.
  • Documentation for the team of designers to reference.
