Explaining the Five Content Types

HTML Text Content Type

The HTML text content type has six width options: 20%, 33%, 50%, 66%, 80%, 100%. Text blocks can be floated to the left or right.

At resolutions under 640px, all text blocks are 100% wide. The floats are arranged based on their order in the HTML document.

Image Content Type

Images have six width options: 20%, 33%, 50%, 66%, 80%, 100%. Image blocks can be floated to the left or right.

At resolutions under 640px, images are centered in their own row and the percentage width is increased.

Video Content Type

Videos have six width options: 20%, 33%, 50%, 66%, 80%, 100%. Video blocks can be floated to the left or right.

At resolutions under 640px, all video blocks are 100% wide. The floats are arranged based on their order in the HTML document.

*Please note, 100% wide is based on the containing element. The "html-banner" container adds 5% padding left and right. To have a video that extends to the edges of a mobile device, place the video outside of the "html-banner" container.

Graphic Divider Content Type

The graphic "divider-wrap" class should be used on images that you want to be 100% width and extend beyond the margins built into the "html-banner" container class. This is suitable for as a visual flourish to logically break up sections or for a promotional banner that links to another page.

Product Content Type

Products have six width options: 20%, 33%, 50%, 66%, 80%, 100%. Product blocks can be floated to the left or right.

At resolutions under 640px, all product blocks are 100% wide. The floats are arranged based on their order in the HTML document.
