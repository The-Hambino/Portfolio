CSS for Visual Content Blocks

/* blank class for assigning content background */ #content-section-ZZ .banner-img-XXX {background-image: url(../images/content-XXX-ZZ.jpg);}

The above CSS is used to assign background images within the "Visual Content Blocks."

"XXX" is to be replaced with the height of the content block section - 650 (px) or 850 (px).

"ZZ" is to be replaced with the a two digit number between "01" - "10". Be sure to add the corresponding id="content-section-ZZ" to the content block section you are adding.

CSS for Hot Spot Links

/* blank classes for defining hot spot dimensions */ #content-section-ZZ.banner-XXX .banner-hot-spot-01 {top: auto; right: auto; bottom: auto; left: auto; width: auto; height: auto;} #content-section-ZZ.banner-XXX .banner-hot-spot-02 {top: auto; right: auto; bottom: auto; left: auto; width: auto; height: auto;}

The above CSS is used to define hot spot dimensions and positions within the "Visual Content Blocks."

Define top/bottom and left/right in percentages. Define width/height in percentages.

Both positioning and dimension percentage values will need to be updated for the mobile version.

The "XXX" and "ZZ" should be replaced with the same values used when defining the background image.

Layout Classes

.html-banner-wrap

Use this class to wrap all custom HTML content.

.html-banner

Use this class to wrap inline image, text and video sections. For any content that should be truly 100% wide (divider images, full-width videos) and touch the edges of the viewport, remove this class.

.row

Use this class to group elements within a single row.

.XXXXX-wrap

.img-wrap, .text-wrap, .product-wrap, .video-box-wrap, .divider-wrap

Use one of these classes to wrap your content based on it's type: inline image, text, inline-product, video or divider image.

.pull-left

Use this class to pull your image, text or video section to the left side of the row.

.pull-right

Use this class to pull your image, text or video section to the right side of the row.

.inline-img-XX

.inline-img-20, .inline-img-33, .inline-img-50, .inline-img-66, .inline-img-80, .inline-img-100

Use one of these classes on the containing wrap of your image to define its width: 20%, 33%, 50%, 66%, 80% or 100%. These percentage widths are for desktop resolutions. Mobile resolutions are optimized for readability and the related CSS is automatic.

.inline-text-XX

.inline-text-20, .inline-text-33, .inline-text-50, .inline-text-66, .inline-text-80, .inline-text-100

Use one of these classes on the containing wrap of your text block to define its width: 20%, 33%, 50%, 66%, 80% or 100%. These percentage widths are for desktop resolutions. Mobile resolutions are optimized for readability and the related CSS is automatic.

.inline-product-XX

.inline-product-20, .inline-product-33, .inline-product-50, .inline-product-66, .inline-product-80, .inline-product-100

Use one of these classes on the containing wrap of your product block to define its width: 20%, 33%, 50%, 66%, 80% or 100%. These percentage widths are for desktop resolutions. Mobile resolutions are optimized for readability and the related CSS is automatic.

.inline-video-XX

.inline-video-20, .inline-video-33, .inline-video-50, .inline-video-66, .inline-video-80, .inline-video-100

Use one of these classes on the containing wrap of your video to define its width: 20%, 33%, 50%, 66%, 80% or 100%. These percentage widths are for desktop resolutions. Mobile resolutions are optimized for readability and the related CSS is automatic.

.divider-img-100

Use this class on the containing wrap of your dividing image to define its width. Dividing images can only be 100% wide.

Helper Classes

.constrain-XXX

.constrain-500, .constrain-600, .constrain-700, .constrain-800, .constrain-900, .constrain-1000

Use this class to constrain the maximum width of the element to the pixel value above.

.spacer-XX

.spacer-10, .spacer-20, .spacer-30, .spacer-40, .spacer-50, .spacer-60, .spacer-70, .spacer-80, .spacer-90, .spacer-100

Use this class to add space before the element. The space added is defined in pixels.
